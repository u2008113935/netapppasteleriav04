# NetApp Pastelería v04

Aplicación móvil para gestión de pedidos de pastelería con .NET MAUI.

## Características
- 📱 Catálogo de productos
- 🛒 Carrito de compras con badge
- 📦 Gestión de pedidos
- 👤 Perfil de usuario
- ⭐ Favoritos
- ⚙️ Configuración

## Tecnologías
- .NET MAUI 8.0
- C# 12
- Supabase
- Shell Navigation con TabBar
